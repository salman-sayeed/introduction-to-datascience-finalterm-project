# Uncovering the Hidden Drivers of Student Depression

### A Data Science Approach Using Lifestyle and Behavioral Patterns

---

## Project Overview

This project is developed as part of the **Introduction to Data Science** course at the **American International University-Bangladesh (AIUB)**, with the long-term goal of publication in a peer-reviewed environmental or data science conference.

This project applies a complete data science pipeline to analyze the underlying factors contributing to student depression. Using a large-scale real-world dataset, the study investigates how academic pressure, financial stress, and lifestyle behaviors interact to influence mental health outcomes.

The objective is not only to build a predictive model but also to extract meaningful insights that explain why depression occurs among students—focusing on identifying hidden, multi-dimensional drivers rather than isolated factors.

The project follows a structured workflow including:

- Data collection and cleaning
- Exploratory Data Analysis (EDA)
- Feature engineering and preprocessing
- Predictive modeling (Logistic Regression)
- Evaluation and interpretation of results

## Dataset Information

The primary dataset used in this research is [`Student Depression Dataset`](https://www.kaggle.com/datasets/hopesb/student-depression-dataset).

### Key Outcome:

The model achieved strong performance in predicting student depression:

- Accuracy: 78.3%
- Precision: 79.5%
= Recall: 83.6%
= F1 Score: 81.5%

The findings reveal that student depression is driven by multiple interacting factors, with academic pressure, financial stress, and workload emerging as the most significant contributors.
